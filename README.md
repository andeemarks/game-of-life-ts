## ➤ Usage

* **test**: yarn lint:types && jest --no-cache
* **lint**: yarn lint:types && yarn lint:ci
* **lint:types**: yarn tsc --noEmit -p .
* **lint:ci**: eslint . --ext .tsx,.ts
* **create**: yarn build
* **game**: yarn build && node lib/src/game-of-life.js
* **build**: tsc -p .
